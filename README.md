# Data Structures and Algorithms (DSA)

This repository contains implementations of various Data Structures and Algorithms concepts in C.

## Topics Covered

### 1. Hashing
Implementation of hashing techniques including:
- Hash table
- Collision handling
- Linear probing
- Separate chaining

### 2. Linked List
Implementation of:
- Singly Linked List
- Doubly Linked List
- Circular Linked List
- Basic operations (insert, delete, search, display)

### 3. Queue
Queue data structure implementations:
- Simple Queue
- Circular Queue
- Priority Queue
- Queue operations using arrays and linked lists

### 4. Searching Techniques
Searching algorithms including:
- Linear Search
- Binary Search

### 5. Sorting Techniques
Sorting algorithms implemented:
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort

### 6. Stack Application
Applications of stack:
- Expression evaluation
- Infix to Postfix conversion
- Parenthesis checking
- Reverse string

### 7. Stack
Stack implementations:
- Stack using Arrays
- Stack using Linked Lists
- Push and Pop operations

### 8. Tree
Tree data structure implementations:
- Binary Tree
- Binary Search Tree (BST)
- Tree Traversals
  - Inorder
  - Preorder
  - Postorder
